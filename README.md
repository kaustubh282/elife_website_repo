# eLife IT Solutions Website

A modern, responsive corporate website developed for **eLife IT Solutions**, showcasing the company's services, expertise, career opportunities, and contact information.

## 🌐 Live Website

- https://www.elifeitsolutions.com
- https://www.elifeitsolutions.com

---

# Features

- Modern responsive UI
- Mobile-friendly design
- Professional navigation
- Home, About, Services, Careers and Contact pages
- Contact Form with EmailJS integration
- Careers Form with EmailJS integration
- Client-side form validation
- Professional favicon and branding
- SEO optimized
- Sitemap and robots.txt
- Custom domain deployment
- SSL secured website

---

# Installation

Clone the repository

```bash
git clone https://github.com/kaustubh282/elife_website_repo.git
```

Go to project folder

```bash
cd elife-website
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Production Build

```bash
npm run build
```

# Contact Form

The Contact page uses **EmailJS** to send enquiries directly to:

```
sales@elifeitsolutions.com
```

Submitted details include:

- Full Name
- Email Address
- Organization
- Mobile Number
- Message

---

# Careers Form

The Careers page collects:

- First Name
- Last Name
- Email Address
- Mobile Number
- Position Applied For
- Additional Message

Application details are sent using EmailJS.

> **Note:** Resume upload is currently a UI feature only. Applicants can be contacted separately to provide their resume via email if shortlisted.
